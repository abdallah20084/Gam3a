# Gam3a5G - منصة تواصل اجتماعي جامعي

منصة تواصل اجتماعي متقدمة مصممة خصيصاً للطلاب الجامعيين، تتيح إنشاء المجموعات والدردشة الجماعية مع دعم الوسائط المتعددة.

## 🚀 المميزات

### 👥 إدارة المجموعات
- إنشاء وإدارة المجموعات الجامعية
- نظام أدوار متقدم (مدير، مشرف، عضو)
- إعدادات خصوصية قابلة للتخصيص
- صور غلاف مخصصة للمجموعات

### 💬 نظام الدردشة
- دردشة فورية في الوقت الحقيقي مع Socket.IO
- دعم الرسائل النصية والوسائط
- إشارة للأعضاء (@mentions)
- ردود على الرسائل
- مؤشرات القراءة والكتابة

### 📱 الوسائط المتعددة
- رفع الصور والفيديوهات
- معاينة الوسائط قبل الرفع
- ضغط الصور تلقائياً
- دعم GIF والملفات المتحركة

### 🔒 الأمان والخصوصية
- نظام مصادقة آمن مع JWT
- تشفير كلمات المرور
- مراقبة المحتوى تلقائياً
- حماية من المحتوى غير اللائق

### 🎨 واجهة مستخدم عصرية
- تصميم متجاوب (Responsive)
- دعم اللغة العربية (RTL)
- واجهة بديهية وسهلة الاستخدام
- سمة داكنة/فاتحة

## 🛠️ التقنيات المستخدمة

### Frontend
- **Next.js 15** - إطار عمل React
- **TypeScript** - لغة البرمجة
- **Bootstrap 5** - إطار CSS
- **Socket.IO Client** - التواصل الفوري
- **React Hook Form** - إدارة النماذج
- **Zod** - التحقق من البيانات

### Backend
- **Custom Server** - خادم مخصص مع Socket.IO
- **Next.js API Routes** - واجهات برمجة التطبيقات
- **MongoDB** - قاعدة البيانات
- **Mongoose** - ODM لـ MongoDB
- **Socket.IO** - خادم WebSocket
- **JWT** - مصادقة المستخدمين
- **bcryptjs** - تشفير كلمات المرور

### Media Processing
- **FFmpeg** - معالجة الفيديو
- **TensorFlow.js** - مراقبة المحتوى
- **NSFW.js** - كشف المحتوى غير اللائق

## 📦 التثبيت والتشغيل

### المتطلبات الأساسية
- Node.js 18+ 
- MongoDB 6+
- npm أو yarn

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone <repository-url>
cd Gam3a5G
```

2. **تثبيت التبعيات**
```bash
npm install
```

3. **إعداد متغيرات البيئة**
```bash
cp env.example .env.local
```

تعديل ملف `.env.local`:
```env
MONGODB_URI=mongodb://localhost:27017/gam3a5g
JWT_SECRET=your-secret-key-here
NEXTAUTH_SECRET=your-nextauth-secret
PORT=3001
```

4. **تشغيل المشروع**

**للتطوير (مع الخادم المخصص):**
```bash
# بناء الخادم أولاً
npm run build-server

# تشغيل الخادم المخصص مع Socket.IO
npm run dev-server
```

**للتطوير (بدون Socket.IO):**
```bash
npm run dev
```

**للإنتاج:**
```bash
# بناء المشروع بالكامل
npm run build
npm run build-server

# تشغيل الخادم
npm run start-server
```

## 🔧 التكوين

### قاعدة البيانات
المشروع يستخدم MongoDB مع Mongoose. تأكد من تشغيل MongoDB قبل تشغيل المشروع.

### الخادم المخصص
المشروع يستخدم خادم مخصص مع Socket.IO للدردشة الفورية:
- المنفذ الافتراضي: 3001
- يدعم WebSocket للتواصل الفوري
- يتكامل مع Next.js API Routes

### الوسائط
يتم تخزين الملفات المرفوعة في مجلد `public/uploads/`. تأكد من وجود صلاحيات الكتابة.

### الأمان
- استخدم مفاتيح سرية قوية لـ JWT_SECRET و NEXTAUTH_SECRET
- قم بتحديث متغيرات البيئة في الإنتاج
- استخدم HTTPS في الإنتاج
