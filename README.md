# Gam3a - Modern Group Chat Application

تطبيق دردشة جماعية حديث مع ميزات متقدمة مستوحاة من أفضل تطبيقات الدردشة العالمية.

## ✨ الميزات الجديدة

### 🎨 تصميم حديث وريسبونسف
- تصميم مستوحى من Discord و Slack و WhatsApp
- واجهة مستخدم عصرية مع تأثيرات بصرية جذابة
- دعم كامل للأجهزة المحمولة والتابلت
- وضع الظلام التلقائي
- تأثيرات حركية سلسة

### 💬 ميزات الدردشة المتقدمة

#### ردود الفعل (Reactions)
- إضافة ردود فعل للرسائل (👍 ❤️ 😂 😮 😢 😡)
- عرض عدد كل رد فعل
- إمكانية إزالة ردود الفعل

#### التعليقات (Comments)
- إضافة تعليقات على الرسائل
- عرض جميع التعليقات مع أسماء المرسلين
- إمكانية الرد على التعليقات

#### الرسائل المثبتة (Pinned Messages)
- تثبيت الرسائل المهمة
- عرض الرسائل المثبتة في الأعلى
- إشعارات للرسائل المثبتة الجديدة
- التنقل السريع للرسائل المثبتة

#### البحث في الرسائل
- البحث في محتوى الرسائل
- تمييز النص المطابق
- عرض عدد النتائج
- التنقل بين النتائج

### 👥 إدارة المستخدمين

#### الملفات الشخصية
- عرض معلومات المستخدم
- صورة الملف الشخصي مع تأثيرات جذابة
- تاريخ الانضمام للمجموعة
- حالة الاتصال (متصل/غير متصل)

#### الرسائل الخاصة
- إرسال رسائل خاصة للمستخدمين
- واجهة منفصلة للرسائل الخاصة
- إشعارات للرسائل الجديدة

#### إدارة الأعضاء (للمديرين)
- كتم صوت المستخدمين
- حظر المستخدمين
- ترقية الأعضاء لمديرين
- طرد الأعضاء من المجموعة

### 🔧 ميزات إضافية

#### تحسينات الواجهة
- أزرار رفع الملفات المحسنة
- اختيار الإيموجي المحسن
- مؤشر الكتابة المتحرك
- شريط التمرير المخصص

#### إمكانية الوصول
- دعم كامل للوحة المفاتيح
- مؤشرات التركيز المحسنة
- دعم وضع التباين العالي
- دعم تقليل الحركة

#### الأداء
- تحميل تدريجي للرسائل
- تحسين الصور والفيديوهات
- تخزين مؤقت للبيانات
- اتصال WebSocket محسن

## 🚀 التثبيت والتشغيل

### المتطلبات
- Node.js 18+
- MongoDB
- Redis (اختياري للتحسين)

### التثبيت
```bash
# استنساخ المشروع
git clone https://github.com/your-username/gam3a.git
cd gam3a

# تثبيت التبعيات
npm install

# إعداد متغيرات البيئة
cp env.example .env.local
# تعديل .env.local بالمعلومات المطلوبة

# تشغيل قاعدة البيانات
npm run db:setup

# تشغيل التطبيق
npm run dev
```

### المتغيرات البيئية
```env
DATABASE_URL="mongodb://localhost:27017/gam3a"
JWT_SECRET="your-secret-key"
NEXTAUTH_SECRET="your-nextauth-secret"
NEXTAUTH_URL="http://localhost:3000"
```

## 🛠️ التقنيات المستخدمة

### Frontend
- **Next.js 14** - إطار العمل الرئيسي
- **React 18** - مكتبة واجهة المستخدم
- **Bootstrap 5** - إطار CSS
- **Socket.io Client** - الاتصال المباشر
- **Emoji Picker React** - اختيار الإيموجي

### Backend
- **Next.js API Routes** - واجهات برمجة التطبيقات
- **Socket.io** - الاتصال المباشر
- **MongoDB** - قاعدة البيانات
- **Prisma** - ORM
- **JWT** - المصادقة

### الأدوات
- **TypeScript** - لغة البرمجة
- **ESLint** - فحص الكود
- **Prettier** - تنسيق الكود
- **Husky** - Git Hooks
